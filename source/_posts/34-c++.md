## Variables 变量

常用的数据类型包括`int`, `double`, `char`, `string`, `bool`

```c++
int age = 23;
double height = 6.1;
char letter = 'X';
string name = 'Yang';
bool isOld = false;

cout << "This is " << name << ", and I am " << age;
```

常量修饰符`const`

```c++
const int MAX_HP = 100;
```

## User Input 用户输入

`cout` 用于 output, `cin`用于 input

```c++
#include <iostream>
using namespace std;

int main() {
    int numOne, numTwo;
    cout << "Enter number one: ";
    cin >> numOne;
    cout << "Enter number two: ";
    cin >> numTwo;
    cout << "The sum is: " << numOne + numTwo;
    return 0;
}

```

## String 字符串

### String Concatenation

Using `+`

```c++
string firstName ="Yang";
string lastName ="Li";

string fullName = firstName + lastName;
```

Or using `.append()` method

```c++
string fullName = firstName.append(lastName);
```

### String Length

Both `length()` and `size()` return the length of the string.

```c++
string msg = "Welcome to c++";
cout << msg.length(); // 15
cout << msg.size(); // 15
```

### Access String

Access specific characts given the index in the string.

```c++
string msg = 'Hello';
char firstLetter = msg[0];
cout << firstLetter; // H
```

Changing String Characters by re-assign the characters

```c++
string firstName = "Yang";
firstName[0] = 'W';
cout << firstName;
```

### User Input `getline()

We can use `cin` but then we can only type a single word, since `cin` see space as termination sign.

Instead, we could use `getline()` to read the line of the input.

```c++
string name;
cout << "What is your name: ";
getline (cin, name);
cout << "Welcome! " << name;
```

## Control Flow

```c++
int wilkScore = 400;

if (wilkScore > 500) {
    cout << "Elite";
} else {
    cout << "Others";
}
```

Or can be written with Ternary Operator

```c++
int wilkScore = 400;
string result = (wilkScore > 500) ? "Elite" : "Beginner";
cout << result;
```

## For Loop

```c++

```

## Break & Continue

With continue, the for loop will end when the action trigger.

```c++
for(int i = 0; i < 10; i++) {
    if(i == 4) {
        break;
    }
    cout << i << "\n";
}
```

However, with continue, the for loop with not execute with that condition, then move on to next iteration and keep looping.

```c++
for(int i = 0; i < 10; i++) {
    if(i == 4) {
        continue;
    }
    cout << i << "\n";
}
```

## Reference

```c++
string food = "Pizza";
string &meal = food;

meal = "Ramen";
cout << meal; // Ramen
cout << food; // Ramen
```

## Pointers

```c++
string food = "Pizza";
string* ptr = &food;    // Create a pointer ptr storing food memory address

cout << food << "\n"; // Pizza
cout << &food << "\n"; // food variable memory address
cout << ptr << "\n"; // ptr which holds food's address
cout << *ptr << "\n"; // the value of the pointer

*ptr = "Ramen"; // reassign the pointer value;

cout << food << "\n"; // Ramen
cout << &food;
```
