event.target与event.currentTarget
发表于 2020-05-28| 更新于 2020-06-20|笔记
阅读量:

JavaScript event.target 和 event.currentTarget 的区别。最近在写一个 react 的项目中，想要通过点击一个元素来触发事件并获得该元素的 key。

event.target

event.target 一般我们可以用来绑定触发事件的元素。比如常用可以在input上来获取文本内容。

比如我们有一个信息对象，可以用event.target来拿到input输入的值然后传到useState里

event.currentTarget

如果我们绑定的是一个元素，比如<i>或者<button>然后想返回触发事件元素的id或者className。

我们可以用event.currentTarget因为这个可以监听当前被触发事件的元素
