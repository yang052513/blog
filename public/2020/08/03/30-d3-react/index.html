Scales

之前的例子中我们用数据给定的数值来定义 svg 元素的高度。假设进行操作的数据值比较大比如 30,000。那么用这个数值来定义元素的高度30000px是不太可行的。我们可以用 D3 中的d3-scale库来对数据的取值范围进行映射并在一个新的范围返回新的对应数值。

具体的步骤为首先利用max找到数据中的最大值，这个值决定了domain。max()第一个参数为所操作的数组，第二个为一个函数即 accessor, 也就是数组的哪个属性我们要找最大值

然后利用scale-linear来创建一个变量函数。domain()和range()的范围默认为从 0 到 1。这里我们设置 domain 为 0 到数组内的最大值。range 即为映射的高度，这里最高设置为 svg 容器的高度。

最后在创建rect元素时，在attr高度里利用上面创建的函数即可。
