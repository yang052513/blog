useEffect(): 管理生命周期

我们之前在 class 组件中可以利用componentDidMount以及其他函数来管理 state 的生命周期。现在，我们可以利用 useEffect 来管理组件的渲染或者处理第三方 API 的请求数据。useEffect()中有两个参数，第一个为 arrow 函数即我们要执行的操作。第二个参数为依赖，当依赖发生变化时就会执行 useEffect。
